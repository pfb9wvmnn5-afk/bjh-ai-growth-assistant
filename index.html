部署说明

这个版本可以直接部署到 GitHub Pages、Netlify、Vercel 或 Cloudflare Pages。 建议文件结构： index.html README.md 手机使用建议： 1. 用 Safari 或微信外部浏览器打开网页。 2. 第一次进入“配置”填写自己的 API Key。 3. 日常新增知识后，定期导出 JSON 备份。 4. 给其他员工同步知识时，让对方导入 JSON。 纯静态边界： 当前版本没有统一云端数据库，不会自动同步每个员工新增的知识。后续可以升级成 Supabase、飞书表格、腾讯文档或 GitHub 文件同步版本。
